## 2. Какие данные мы собираем

| Данные | Источник | Цель |
|--------|----------|------|
| Telegram ID | Telegram API | Идентификация пользователя |
| Имя пользователя (username) | Telegram API | Отображение в аналитике |
| Дата первого входа | Наша система | Учёт подписки |
| Факты активности: дата/время вступления и выхода, количество сообщений за период, количество реакций за период | Telegram API (бот-администратор) | Аналитика рефералов |
| Данные о платежах | ЮKassa | Учёт подписки |

2.1. Мы **не собираем**: номер телефона, email, геолокацию, содержимое сообщений.

2.2. Данные об активности собираются **только в тех чатах**, где владелец канала сам добавил бота как администратора.

## 3. Правовые основания обработки

- **Исполнение договора** (ст. 6 GDPR, ст. 6 152-ФЗ) — для предоставления услуг Сервиса.
- **Законные интересы** — для предотвращения мошенничества и накруток.
- **Согласие** — при первом запуске бота пользователь получает уведомление об обработке данных и нажимает кнопку «Принять». Только это действие является явным согласием. При отказе функции бота недоступны.

---

## 4. Как мы используем данные

- Предоставление функций Сервиса (аналитика, отчёты).
- Антифрод-проверки (выявление ботов и накруток).
- Учёт и обработка платежей.
- Отправка уведомлений о подписке.
- Улучшение качества Сервиса (агрегированная статистика без персонализации).

## 7. Ваши права

- **Получить** копию данных — команда `/export` (JSON).
- **Исправить** данные — через поддержку.
- **Удалить** данные — команда `/delete_account`. Анонимизация в течение 30 дней (кроме платежей — 5 лет). Логи активности удаляются немедленно.
- **Отозвать согласие** — команда `/revoke_consent`. Отзыв согласия **не равнозначен** удалению данных; для удаления используйте `/delete_account`.
- **Ограничить обработку** (ст. 18 GDPR) — через поддержку.
- **Перенести данные** (ст. 20 GDPR) — команда `/export`.
- **Пожаловаться** в Роскомнадзор (РФ) или надзорный орган по месту жительства (ЕС).

---

## 8. Безопасность

- Данные хранятся с шифрованием в покое (AES-256), передача — по HTTPS/TLS.
- Доступ к данным ограничен минимально необходимым кругом лиц.
- При утечке данных:
  - Роскомнадзор уведомляется в течение **24 часов** (152-ФЗ, ред. 2022).
  - Пользователи уведомляются в течение **72 часов** (ст. 33 GDPR, для пользователей из ЕС).
