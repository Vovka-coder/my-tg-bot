# RefLens — UX/UI Design Document v2.0

## Принципы проектирования

1. **Минимум кликов** — любое ключевое действие не более 3-4 шагов
2. **Два типа клавиатур** — Reply (главное меню, всегда видно) + Inline (контекстные действия)
3. **Прогрессивное раскрытие** — не показывать всё сразу, вести за руку
4. **Мобильная адаптация** — макс. 2 кнопки в ряд, макс. 20 символов на кнопку

---

## Главное меню (Reply-клавиатура)

Появляется после завершения онбординга. Всегда доступна.

```
[📈 Аналитика]      [🌳 Дерево связей]
[⭐ Топ рефереров]  [⚙️ Настройки]
[💎 Подписка]       [🆘 Поддержка]
```

## Команды бота

```
/start          — запуск / онбординг
/menu           — главное меню
/connect        — подключить канал
/stats          — аналитика (последние 7 дней)
/tree           — дерево рефералов
/top            — топ-10 рефереров
/settings       — настройки бота и правил
/tariffs        — тарифы и подписка
/export         — экспорт данных (CSV)
/support        — поддержка
/privacy        — политика конфиденциальности
/cancel         — отменить подписку
/freeze         — заморозить подписку
/delete_account — удалить аккаунт и данные
/revoke_consent — отозвать согласие на обработку данных
/whats_new      — что нового в боте
```

## 1️⃣ Первый запуск и онбординг

**Цель:** зарегистрировать, получить согласие, подключить канал, настроить правила.

| Шаг | Действие пользователя | Действие бота | Кнопки | Клики |
|-----|-----------------------|---------------|--------|-------|
| 1 | /start | Приветствие + ценность бота | [Начать 👋] [Узнать больше] | 1 |
| 2 | Нажимает «Начать» | Уведомление о данных + запрос согласия | [Принять и продолжить] [Подробнее о данных] | 2 |
| 3 | Принимает | Инструкция по добавлению бота в канал | [Добавить в канал →] [Я добавил, проверить] | 3 |
| 4 | Добавил бота → «Проверить» | Проверяет права → успех или инструкция по ошибке | [Настроить правила ⚙️] [В главное меню] | 4 |
| 5 | «Настроить правила» | Мин. возраст аккаунта для реферала | [7 дней] [14 дней] [30 дней] | 5 |
| 6 | Выбирает возраст | Мин. активность для реферала | [1 сообщение] [3 сообщения] [Только подписка] | 6 |
| 7 | Выбирает активность | Мини-тур по командам + главное меню | Reply-клавиатура | 7 |

**Итого: 7 кликов + внешнее действие (добавление бота)**

**Текст шага 2 (согласие):**
```
Для работы мне нужны данные из твоего канала:
• Кто и когда вступил/вышел
• Количество сообщений и реакций участников

Я не читаю личные сообщения и не передаю данные третьим лицам.
Подробнее: /privacy

[Принять и продолжить]  [Подробнее о данных]
```

## FSM — состояния пользователя

```
NEW_USER
  └─→ CONSENT_PENDING     (ждём согласие)
        └─→ ONBOARDING     (подключение канала)
              └─→ SETUP_RULES (настройка антифрода)
                    └─→ ACTIVE (основной режим)

ACTIVE
  ├─→ VIEWING_STATS
  ├─→ VIEWING_TREE
  ├─→ CHECKOUT
  └─→ CANCELLATION_FLOW

SUBSCRIPTION:
  FREE → TRIAL → PRO/BUSINESS
  PRO/BUSINESS → FROZEN
  PRO/BUSINESS → FREE
```

---

## Правила мобильной адаптации

- Максимум **2 кнопки в ряд**
- Длина кнопки — не более **20 символов**
- Длина сообщения — не более **5 строк** основного текста
- Важные цифры — **в начале** сообщения, не в конце
